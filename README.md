# Felipe's Blog

Personal project to practice React Router library and some other core concepts of React itself.

## Technologies
- React 19
- React Router 7
- Vite

## Practical concepts
- Dynamic routes
- Layouts
- Lazy loading

## How to run

### Pre-requisites
- Node.js (v22)
- npm

### Installation
```bash
git clone https://github.com/felipe-u/felipes-blog.git
cd felipes-blog
npm install
```

### Run
```bash
npm run dev
```
